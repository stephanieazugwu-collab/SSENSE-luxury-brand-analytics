# SSENSE Luxury Brand Analytics

## Overview
This project analyzes luxury brand pricing and category trends on SSENSE using Python.
The goal is to uncover how brand positioning, product type, and gender influence pricing.

## Dataset
- Source: Kaggle – SSENSE Fashion Dataset
- Format: CSV
- Size: ~12,000 product listings

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- Regex
- Jupyter Notebook

## Key Analyses
- Top designers by average price
- Gender-based pricing comparison
- Category-level pricing trends
- Price distribution across top brands
- Interactive brand insight tool

## Files
- `ssense_analysis.ipynb` – Full analysis notebook
- `ssense_analysis.html` – Rendered notebook for easy viewing
- `SSENSE_Luxury_Analytics_Report.pdf` – Final written report
